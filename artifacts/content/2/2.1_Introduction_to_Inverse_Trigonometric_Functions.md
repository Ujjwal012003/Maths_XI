### 2.1.1 Definition of Inverse Trigonometric Functions

The **Inverse Trigonometric Functions** are functions which "reverse" the action of the basic trigonometric functions (sine, cosine, tangent, cotangent, secant, cosecant). In other words, if a trigonometric function gives the ratio of sides for a given angle, its inverse function returns the angle corresponding to a given ratio.

Formally, if $y = \sin x$, then the inverse sine function is defined as $x = \sin^{-1} y$, where $\sin^{-1}$ is read as "inverse sine" or "arcsine".

#### Geometric Meaning (Graphical Interpretation)

Consider the sine function defined on the interval $\left[-\frac{\pi}{2}, \frac{\pi}{2}\right]$, where it is strictly increasing and hence one-to-one. Restricting sine to this domain, the inverse function $\sin^{-1} x$ is defined as the function mapping $y$ back to $x$. Its graph is a reflection of $y=\sin x$ about the line $y=x$.

This restriction is crucial to make the inverse meaningful because trigonometric functions are periodic and not one-to-one on their entire domains.

---

### 2.1.2 Why Restrict Domain of Trigonometric Functions?

The primary reason for restricting the domain of trigonometric functions before defining their inverses is to ensure **bijectivity** (both injective and surjective nature, i.e., one-to-one and onto), because only then does an inverse function exist.

For example, consider $y = \sin x$. On the entire real line, sine is periodic and many values of $x$ produce the same $y$. Hence, it is not one-to-one, and strict inverses do not exist unless the domain is restricted to an interval where sine is strictly monotonic and covers its entire range $[-1,1]$ uniquely, which is $\left[-\frac{\pi}{2}, \frac{\pi}{2}\right]$.

Similar domain restrictions apply to other functions:

| Function | Restricted Domain                 | Range            |
| -------- | ------------------------------- | ---------------- |
| $\sin x$ | $\left[-\frac{\pi}{2}, \frac{\pi}{2}\right]$ | $[-1,1]$         |
| $\cos x$ | $[0, \pi]$                      | $[-1,1]$         |
| $\tan x$ | $\left(-\frac{\pi}{2}, \frac{\pi}{2}\right)$ | $\mathbb{R}$ (all real numbers) |

This restriction guarantees the existence of an inverse function, which can then "undo" the trigonometric function uniquely.

### 2.1.3 Notation of Inverse Trigonometric Functions and Basic Examples

The inverse trigonometric functions are commonly denoted as:

| Trigonometric Function | Inverse Function Notation | Also Called      |
| --------------------- | ------------------------ | ---------------- |
| Sine                  | $\sin^{-1} x$ or $\arcsin x$ | Arcsine          |
| Cosine                | $\cos^{-1} x$ or $\arccos x$ | Arccosine        |
| Tangent               | $\tan^{-1} x$ or $\arctan x$ | Arctangent       |
| Cotangent             | $\cot^{-1} x$ or $\arccot x$ | Arccotangent     |
| Secant                | $\sec^{-1} x$ or $\arcsec x$ | Arcsecant        |
| Cosecant              | $\csc^{-1} x$ or $\arccsc x$ | Arccosecant      |

**Note:** The superscript $-1$ denotes inverse function and is not to be confused with reciprocal.

**Theorem 1: Existence of Inverse Trigonometric Functions**

\[
\boxed{
\begin{aligned}
&\text{If } f: D \to R \text{ is one-to-one and onto, then}\\
&\text{there exists a function } f^{-1} : R \to D \text{ such that } f^{-1}(f(x)) = x \text{ for all } x \in D.
\end{aligned}
}
\]

**Proof:**  

1. Let $f$ be one-to-one (injective), so that if $f(x_1) = f(x_2)$, then $x_1 = x_2$.

2. Let $f$ be onto (surjective), so for every $y \in R$, there exists at least one $x \in D$ such that $f(x) = y$.

3. Define $f^{-1}(y) = x$ as the unique element in $D$ with $f(x) = y$ (uniqueness follows from injectivity).

4. Then, for all $x \in D$,
\[
f^{-1}(f(x)) = x
\]
and for all $y \in R$,
\[
f(f^{-1}(y)) = y,
\]
thus $f^{-1}$ is the inverse function of $f$.

### 2.1.4 Formal Definitions of Inverse Trigonometric Functions

For $x$ in appropriate domains, the inverse trigonometric functions are defined by these relations:

1. **Inverse Sine Function**: If $y = \sin^{-1} x$, then
\[
\sin y = x, \quad y \in \left[-\frac{\pi}{2}, \frac{\pi}{2}\right], \quad x \in [-1,1].
\]

2. **Inverse Cosine Function**: If $y = \cos^{-1} x$, then
\[
\cos y = x, \quad y \in [0, \pi], \quad x \in [-1,1].
\]

3. **Inverse Tangent Function**: If $y = \tan^{-1} x$, then
\[
\tan y = x, \quad y \in \left(-\frac{\pi}{2}, \frac{\pi}{2}\right), \quad x \in \mathbb{R}.
\]

4. **Inverse Cotangent Function**: If $y = \cot^{-1} x$, then
\[
\cot y = x, \quad y \in (0, \pi), \quad x \in \mathbb{R}.
\]

5. **Inverse Secant Function**: If $y = \sec^{-1} x$, then
\[
\sec y = x, \quad y \in \left[0, \pi\right] \setminus \left\{\frac{\pi}{2}\right\}, \quad |x| \geq 1.
\]

6. **Inverse Cosecant Function**: If $y = \csc^{-1} x$, then
\[
\csc y = x, \quad y \in \left[-\frac{\pi}{2}, \frac{\pi}{2}\right] \setminus \{0\}, \quad |x| \geq 1.
\]

### 2.1.5 Formulas and Working Rule (Algorithm)

**Basic Formulas:**

\[
\sin(\sin^{-1} x) = x, \quad x \in [-1,1],
\]
\[
\cos(\cos^{-1} x) = x, \quad x \in [-1,1],
\]
\[
\tan(\tan^{-1} x) = x, \quad x \in \mathbb{R}.
\]

Similarly,

\[
\sin^{-1}(\sin x) = x, \quad x \in \left[-\frac{\pi}{2}, \frac{\pi}{2}\right],
\]
\[
\cos^{-1}(\cos x) = x, \quad x \in [0, \pi],
\]
\[
\tan^{-1}(\tan x) = x, \quad x \in \left(-\frac{\pi}{2}, \frac{\pi}{2}\right).
\]

---

**Working Rule (Algorithm) to Solve Inverse Trigonometric Problems**

1. **Step 1:** Identify the inverse trigonometric function involved and the given value.

2. **Step 2:** Check the **domain and range restrictions** relevant to that inverse function to ensure the input value is valid.

3. **Step 3:** Use the defining equation of inverse functions such as $\sin y = x$ for $y = \sin^{-1} x$ with the restricted domain of $y$ to find the value of $y$.

4. **Step 4:** If the problem requires evaluating composite functions like $\sin(\sin^{-1} x)$ or involving expressions, apply the basic formulas using domain knowledge.

5. **Step 5:** For solving equations or simplifying expressions, consider the range of the inverse trig function to select the principal value (the main value in the restricted domain).

6. **Step 6:** Verify the solution graphically or by substitution if necessary to confirm it lies within the defined domain/range.

### 2.1.6 Graded Solved Examples

#### Type 1: Concept Building (Basic Direct Application)

**Example 1:**

Find the value of $\sin^{-1} \left(\frac{1}{2}\right)$.

**Solution:**

- **Step 1:** Let $y = \sin^{-1} \left(\frac{1}{2}\right)$.

- **Step 2:** By definition, $\sin y = \frac{1}{2}$ and $y \in \left[-\frac{\pi}{2}, \frac{\pi}{2}\right]$.

- **Step 3:** From known values, $\sin \frac{\pi}{6} = \frac{1}{2}$.

- **Step 4:** Since $\frac{\pi}{6} \in \left[-\frac{\pi}{2}, \frac{\pi}{2}\right]$, the principal value is $y = \frac{\pi}{6}$.

**Answer:**

\[
\sin^{-1} \left(\frac{1}{2}\right) = \frac{\pi}{6}.
\]

---

#### Type 2: Competency Based (Intermediate/Tricky)

**Example 2:**

Find the value of $\cos^{-1} \left(\cos \frac{7\pi}{6}\right)$.

**Solution:**

- **Step 1:** The expression is $\cos^{-1}(\cos \theta)$, with $\theta = \frac{7\pi}{6}$.

- **Step 2:** The principal range of $\cos^{-1} x$ is $[0, \pi]$.

- **Step 3:** Since $\frac{7\pi}{6} = \pi + \frac{\pi}{6}$ lies outside the range $[0, \pi]$.

- **Step 4:** Use the property:

For $x \notin [0, \pi]$, 
\[
\cos^{-1}(\cos x) =
\begin{cases}
x, & x \in [0, \pi], \\
2\pi - x, & x \in [\pi, 2\pi].
\end{cases}
\]

- **Step 5:** Since $\frac{7\pi}{6} \in (\pi, 2\pi)$,
\[
\cos^{-1} \left(\cos \frac{7\pi}{6}\right) = 2\pi - \frac{7\pi}{6} = \frac{5\pi}{6}.
\]

- **Step 6:** Note that $\frac{5\pi}{6} \in [0, \pi]$, so this is the principal value.

**Answer:**

\[
\cos^{-1} \left(\cos \frac{7\pi}{6}\right) = \frac{5\pi}{6}.
\]

---

#### Type 3: HOTS (Master Problem)

**Example 3:**

Solve the equation 
\[
\sin^{-1} x + \cos^{-1} x = \frac{\pi}{2}
\]
for $x$.

**Solution:**

- **Step 1:** We are given the sum of inverse sine and inverse cosine.

- **Step 2:** Use the identity (proved below or known):
\[
\sin^{-1} x + \cos^{-1} x = \frac{\pi}{2}, \quad x \in [-1,1].
\]

- **Step 3:** Check for which $x$ it holds. The identity is true for all $x \in [-1,1]$ by the theorem.

- **Step 4:** Therefore, the equation holds for all $x$ in $[-1,1]$.

**Answer:**

\[
\boxed{\text{All } x \in [-1,1]}.
\]

---

### 2.1.7 Theorem: Relationship Between Inverse Sine and Inverse Cosine

\[
\boxed{
\sin^{-1} x + \cos^{-1} x = \frac{\pi}{2}, \quad \forall x \in [-1,1].
}
\]

**Proof:**

- **Step 1:** Let $y = \sin^{-1} x$, so that $\sin y = x$ and $y \in \left[-\frac{\pi}{2}, \frac{\pi}{2}\right]$.

- **Step 2:** Let $z = \cos^{-1} x$, so that $\cos z = x$ and $z \in [0, \pi]$.

- **Step 3:** Using the identity $\cos z = \sin \left(\frac{\pi}{2} - z\right)$, we have
\[
\sin y = x = \cos z = \sin \left(\frac{\pi}{2} - z\right).
\]

- **Step 4:** Since $y \in \left[-\frac{\pi}{2}, \frac{\pi}{2}\right]$ and $\frac{\pi}{2} - z \in \left[-\frac{\pi}{2}, \frac{\pi}{2}\right]$, and sine is one-to-one on this interval, equate arguments:
\[
y = \frac{\pi}{2} - z.
\]

- **Step 5:** Rearranging,
\[
y + z = \frac{\pi}{2},
\]
i.e.,
\[
\sin^{-1} x + \cos^{-1} x = \frac{\pi}{2}.
\]

This completes the proof.
